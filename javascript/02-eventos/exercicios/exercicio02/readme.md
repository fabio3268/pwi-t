## Cadastro de Usuário

- Conforme o usuário digita seu nome no respectivo campo, o mesmo deve ser exibido em tempo real na divName.
- Depois que a data do nascimento for informada, a divBirth deve ser preenchida automaticamente com a idedade.
- Conforme o usuário digita o Endereço, o mesmo deve ser exibido em tempo real na divAddress.
